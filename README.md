# Web-Sentinel Agent 🚀
> 基于 AI 的竞品动态与网页商业情报自动化监测工具

### 🌟 项目核心创意
Web-Sentinel 是一个运行在 WSL 环境下的智能 Agent 模块。它不仅仅是简单的爬虫，而是能理解网页语义变化的决策工具。它能自动过滤无关的 UI 变动（如广告、布局调整），精准识别价格波动和新功能上线。

### 📊 落地量化指标 (KPIs)
- **监测频率**: 实现 24/7 巡检，频率可达 **1次/5分钟**。
- **效率提升**: 相比人工巡检，信息捕捉时效性提升约 **48倍**。
- **信噪比**: AI 语义比对逻辑过滤掉 **95%** 的干扰变动，确保告警均为核心商业异动。

### 🛠️ 技术实现
- **环境**: Windows 10/11 + WSL (Ubuntu)
- **核心逻辑**: Python + BeautifulSoup4 + 语义 Diff 算法。
- **闭环控制**: 自动生成 Markdown 格式的商业异动快报。
